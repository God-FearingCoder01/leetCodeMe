## 📌 About

**leetCodeMe** is my solo coding practice repository where I solve LeetCode problems ( or at least try 😆) primarily using Python.

This repo serves as:

- A log of my progress and consistency
- A reference for problem-solving patterns and Python techniques
- A personal archive of optimized and well documented solutions

Whether you're here for inspiration, collaboration or just curiosoty - you're welcome! 🚀
